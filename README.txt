Brands the client dashboard

== Description ==

This plugin is intended for Lime Canvas customers only.

It customises the login logo, removes default WordPress dashboard widgets and the WordPress admin menu logo.

It adds a Lime Canvas News dashbaord widget which gets the RSS feed from the Lime Canvas website.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `lc-branding.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
